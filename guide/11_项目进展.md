## 一、项目概述
项目 TestCaseAgent - 基于 LangGraph 的智能测试用例生成 Agent 核心目标 输入自然语言描述，自动生成可执行的 pytest 测试用例 当前阶段 Alpha 0.x - 核心流程可运行，但不稳定

## 二、已完成的功能 ✅
### 1. 核心架构
模块 状态 说明 LangGraph 状态图 ✅ 完成 5 节点线性流程 节点编排 ✅ 完成 requirement_parser → context_retriever → planner → generator → sandbox_executor 记忆管理 ✅ 完成 Checkpointer + InMemoryStore CLI 界面 ✅ 完成 Rich Tree 实时显示 + 耗时统计

### 2. 核心功能

```
1. 先写测试，再实现
2. 最小实现，确保通过
3. 重构，保持测试通过
```

#### 原则2：Mock 先行
```python
from unittest.mock import Mock, patch

@pytest.fixture
def mock_llm():
    """Mock LLM 响应"""
    mock = Mock()
    mock.invoke.return_value = Mock(content="test response")
    return mock
```

- 每个测试独立运行
- 不依赖真实网络
- 不依赖真实 API 密钥
- 不依赖外部服务
