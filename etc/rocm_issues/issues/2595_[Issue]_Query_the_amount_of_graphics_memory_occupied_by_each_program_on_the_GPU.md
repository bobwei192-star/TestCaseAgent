### 评论 #4 — harkgill-amd (2024-07-17T17:41:44Z)

Hi @taikai-zz, could you please try the `rocm-smi --showpids` command with the latest ROCm 6.1.2 release? A small fix was introduced in ROCm 6.1.0. 

"Fixed `--showpids` reporting `[PID] [PROCESS NAME] 1 UNKNOWN UNKNOWN UNKNOWN`.
Output was failing because `cu_occupancy debugfs` method is not provided on some graphics cards
by design. `get_compute_process_info_by_pid` was updated to reflect this and returns with the output
needed by the CLI."

More information can be found [here](https://rocm.docs.amd.com/projects/rocm_smi_lib/en/latest/CHANGELOG.html#id8).
