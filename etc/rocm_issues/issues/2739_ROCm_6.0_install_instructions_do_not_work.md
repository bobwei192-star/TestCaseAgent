The HIP SDK for ROCm was released for Windows in July, allowing Windows users to use various features like rocBLAS on Windows, so while full ROCm support isn't released yet, ROCm features are available on Windows. In an upcoming ROCm release there will be MiOpen support for Windows too, the commit to build AMD's Machine Intelligence Library on Windows was released the other day https://github.com/ROCm/MIOpen/commit/ffe4f8d24faae71d8a8d5e08d073a8c1281e2ae8



> Hello, I have recently failed to install ROCm on my windaos10 system. Can you tell me how to install it? Thank you very much.

The main installation guide is here: https://rocm.docs.amd.com/projects/install-on-windows/en/latest/
All you basically need to do is install this software: https://www.amd.com/en/developer/resources/rocm-hub/hip-sdk.html

Make sure you have a compatible GPU: https://rocm.docs.amd.com/projects/install-on-windows/en/latest/reference/system-requirements.html

### 评论 #9 — harkgill-amd (2024-08-15T15:08:23Z)

Hi @baryluk, the install instructions have since been fixed for ROCm 6.0.0 and onwards. At the [Ubuntu native installation](https://rocm.docs.amd.com/projects/install-on-linux/en/docs-6.0.0/how-to/native-install/ubuntu.html) page for ROCm 6.0.0 
1. The unnecessary `done` was removed.
2. repo links were updated to the correct `6.0` paths. 
3. `sudo apt update` was added at the end of the [Register ROCm packages](https://rocm.docs.amd.com/projects/install-on-linux/en/docs-6.0.0/how-to/native-install/ubuntu.html#register-rocm-packages) code block.

Thanks for all your help!
